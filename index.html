🛣️ Road Classification with U-Net

Segmenting roads from aerial imagery using deep learning

🚀 Overview

This project uses a U-Net model for semantic segmentation to classify road pixels in high-resolution satellite imagery, developed as part of a research initiative in October 2024.

📊 Key Features

  • Model: U-Net architecture (PyTorch)
  • Dataset: DeepGlobe Road Extraction
  • Task: Binary road vs non-road segmentation
  • Metrics: IoU, F1 Score

🧠 Architecture

Inspired by the original U-Net paper for biomedical segmentation:

🔗 U-Net: Convolutional Networks for Biomedical Image Segmentation
